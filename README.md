## Рейтинг
1. Formik - Лучший для React с полным управлением состоянием форм, идеален для сложных сценариев
2. vee-validate - Топ для Vue.js с отличной TypeScript поддержкой и композиционным API
3. AJV - Максимально быстрый валидатор JSON-данных, но не для UI-форм
4. validate.js - Универсальная легковесная библиотека, но без современной интеграции с фреймворками
5. PristineJS - Супер-простая валидация для базовых нужд, но очень ограниченный функционал

## JustValidate
Точка входа: `import JustValidate from 'just-validate';`

### Минимальный необходимый набор методов
- `new JustValidate(form, options)` - создание валидатора
- `.addField(selector, rules)` - добавление поля с правилами
- `.onSuccess(callback)` - обработка успешной валидации
- `.onFail(callback)` - обработка ошибок валидации

### Часто используемые правила валидации
- `{ rule: 'required' }` - обязательное поле
- `{ rule: 'email' }` - валидация email
- `{ rule: 'minLength', value: 3 }` - минимальная длина
- `{ rule: 'maxLength', value: 20 }` - максимальная длина
- `{ rule: 'number' }` - только числа
- `{ rule: 'password' }` - проверка пароля

### Дополнительные полезные методы
- `.revalidate()` - принудительная перепроверка
- `.destroy()` - удаление валидатора
- `.setCurrentLocale(locale)` - локализация ошибок

### Типы данных
- `form` - string (селектор) или DOM-элемент
- `rules` - массив объектов с правилами
- `options` - объект с настройками (подсказки, фокус и т.д.)
